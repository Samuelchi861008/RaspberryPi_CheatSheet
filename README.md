## 機體資訊
  * 型號 : Raspberry Pi 4B
  * Ram : 4G (同款有 2G 與 8G 可供選擇)

## 必備項目
  * 主機體
  * 64G SD記憶卡 
  * Type-C 變壓器 (建議購置原廠)
  * 保護機體外殼 (含風扇和散熱片)
  * HDMI(母) 轉 Micro HDMI(公) 轉接頭
  * SD記憶卡讀卡器
  * 其他依據自己應用需求採購感應器與線材
  
  採購參考網頁：https://www.eclife.com.tw/led/0703300039/1401100001/160309001
